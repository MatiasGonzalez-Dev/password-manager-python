# Password Manager Python

Proyecto personal desarrollado para aprender Python desde cero mediante la construcción de un gestor de cuentas y credenciales.

## Objetivos

- Aprender programación estructurada.
- Practicar funciones, condicionales y ciclos.
- Aprender Git y GitHub.
- Construir un gestor de cuentas y credenciales de uso personal.
- Aplicar buenas prácticas de desarrollo de software.

## Funcionalidades actuales

### Autenticación

- Login básico con usuario y contraseña.
- Validación de credenciales.
- Control de acceso al sistema.

### Navegación

- Menú principal interactivo.
- Menús independientes por categoría.
- Retorno al menú principal mediante `return`.
- Navegación continua mediante `while`.
- Salida controlada mediante `break`.

### Categorías disponibles

- Bancos
- Brokers
- Criptomonedas
- Correos
- Redes sociales
- Notas

## Tecnologías utilizadas

- Python 3
- Git
- GitHub

## Cómo ejecutar el proyecto

```bash
python3 main.py
```
