# Password Manager Python

Proyecto personal desarrollado para aprender Python desde cero.

## Objetivos

- Aprender programación estructurada.
- Practicar funciones, listas y diccionarios.
- Aprender Git y GitHub.
- Construir un gestor de cuentas y credenciales de uso personal.

## Funcionalidades actuales

- Login básico.
- Menú principal.
- Menú de bancos.
- Menú de brokers.
- Menú de criptomonedas.
- Menú de correos.
- Menú de redes sociales.
- Menú de notas.
